# 💁 Customer Service

Customer Service is a simple spring boot REST api that serves you a list of customers!

## Model
### Customer  
|  attribute  |  type   | required  |
|-------------|---------|-----------|
| firstName   | String  |    Yes    |
| lastName    | String  |    Yes    |
| companyName | String  |    Yes    |

## Endpoints
* GET v1/customers/
  * Returns a list of Customer objects
